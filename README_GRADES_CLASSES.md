# Grades and Classes - Kenyan Education System

## Understanding the System

In Kenya, the education system follows this structure:

### Grades (Academic Levels)
- **PP1** - Pre-Primary 1
- **PP2** - Pre-Primary 2
- **G1-G6** - Grade 1 to Grade 6 (Primary)
- **G7-G9** - Grade 7 to Grade 9 (Junior Secondary School - JSS)

### Classes (Physical Classrooms)

In most Kenyan schools:
- **One class per grade**: The class name is often the same as the grade (e.g., "PP1", "Grade 1")
- **Multiple classes per grade**: Some large schools have multiple classes per grade (e.g., "PP1 A", "PP1 B", "Grade 1 A", "Grade 1 B")

## How to Set Up

### Option 1: One Class Per Grade (Most Common)
1. Create grades: PP1, PP2, G1, G2, etc.
2. For each grade, create a class with the same name:
   - Grade: PP1 → Class: "PP1"
   - Grade: Grade 1 → Class: "Grade 1"
   - Grade: Grade 7 (JSS) → Class: "Grade 7"

### Option 2: Multiple Classes Per Grade (Large Schools)
1. Create grades: PP1, PP2, G1, G2, etc.
2. For each grade, create multiple classes:
   - Grade: PP1 → Classes: "PP1 A", "PP1 B", "PP1 C"
   - Grade: Grade 1 → Classes: "Grade 1 A", "Grade 1 B"

## Best Practices

1. **Create grades first** before creating classes
2. **Use consistent naming**: If using single class per grade, use the grade name as class name
3. **Assign class teachers** to manage each class
4. **Set appropriate capacity** based on your school's needs (typically 30-45 students)

## Example Setup

### Large School (Multiple classes per grade):

- ... and so on

## Academic Year Format

Use format: **YYYY/YYYY** (e.g., 2024/2025)
- First year: Start of academic year
- Second year: End of academic year
